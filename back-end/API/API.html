Metadata

Up 
get /like
retrieves likes for a document/page (getLikes)
Gets the number of likes of a given document/page and tells if the user who did the request liked it or not

Query parameters

id (required)
Query Parameter — pass the id of the required document/page
user (required)
Query Parameter — user who did the request format: uuid
type (required)
Query Parameter — says if it is a document or a page (doc = 0, page = 1) format: int32

Return type

GetLikes

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

likes obtained GetLikes

400

bad input parameter

500

internal error
